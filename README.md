# SifaPedal

SifaPedal maps a physical steering wheel pedal to a customized keyboard stroke, turning it into a functional Sifa
(dead-man's switch) pedal for PC train simulators.
To send an acknowledgment, it requires a full cycle of lifting and repressing the pedal, accurately mimicking real-world
Sifa mechanics. 
Additionally, it features an optional emergency brake system that automatically activates if the pedal is released and
not repressed within a specified time limit.
